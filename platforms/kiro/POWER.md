# HCLS Agents Toolkit Power

This power provides domain knowledge for building healthcare and life sciences AI agents on AWS.

## Capabilities

- Guide agent development for genomics, drug discovery, clinical trials, biomarker analysis
- Connect to biomedical databases, ontology services, and literature search via MCP
- Reference 36+ production-ready agents in the catalog
- Deploy agents to Amazon Bedrock AgentCore (Runtime, Gateway, Memory, Registry)

## When to Activate

- User is building an HCLS agent or workflow
- User asks about genomics, drug discovery, clinical trials, or biomarker analysis
- User wants to deploy agents to AgentCore
- User needs to connect to biomedical data sources
